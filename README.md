# Library Management System

## 📚 Project Overview
The Library Management System is a Core Java application developed as part of the Winter Internship Program. It helps manage library operations such as adding books, displaying available books, issuing books, and returning books through a menu-driven console application.

## ✨ Features
- Add New Books
- Display Available Books
- Issue Books
- Return Books
- Book Availability Status
- Menu-Driven Console Interface

## 🛠 Technologies Used
- Core Java
- Object-Oriented Programming (OOP)
- Collections Framework (ArrayList)
- Interfaces
- Abstract Classes
- Exception Handling
- Visual Studio Code
- Git & GitHub

## ▶️ How to Run
1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Compile the project:
   ```
   javac *.java
   ```
4. Run the project:
   ```
   java Main
   ```
